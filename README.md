# SimpleDictionary

This is a simple Android dictionary app made using **Java and XML**.

## What the app does

- Shows a list of words and meanings.
- Uses `RecyclerView` to display the list.
- Uses a `FloatingActionButton` to add a new word.
- Has another screen called `AddEditActivity`.
- The second screen has two input fields:
    - Word
    - Meaning
- The user can save a new word.
- The user can click on a word to edit it.
- The user can long click on a word to delete it.
- Toast messages appear when adding, editing, deleting, or leaving fields empty.
- The app keeps the data when the screen rotates.
- The app also saves the data after closing and opening it again.

## Main files

- `MainActivity.java`
- `AddEditActivity.java`
- `WordAdapter.java`
- `activity_main.xml`
- `activity_add_edit.xml`
- `item_word.xml`

## How to run the app

1. Open the project in Android Studio.
2. Wait for Gradle sync to finish.
3. Run the app on an emulator or real Android phone.
